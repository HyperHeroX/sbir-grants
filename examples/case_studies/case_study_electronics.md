# 電子電機產業 SBIR 案例指南

> 電子電機是台灣強項，也是 SBIR 熱門申請領域。

## 產業概述

### 市場規模

| 指標 | 數據 |
|------|------|
| 台灣電子業產值 | 約 5.8 兆元（2024） |
| 年成長率 | 3-5% |
| 全球市佔 | 半導體代工 60%+、PCB 30%+ |

### SBIR 電子產業申請趨勢

電子電機類申請案約佔總申請案 **20-25%**，競爭激烈但通過率與整體平均相近。

## 成功案例分析

### 案例一：AI 光學檢測系統

**公司背景**：
- 影像處理軟體公司
- 成立 8 年，員工 35 人
- 專精工業視覺技術

**計畫內容**：
> 開發深度學習驅動的 AOI（自動光學檢測）系統，適用於 PCB 及半導體封裝

**成功要素**：

| 要素 | 做法 |
|------|------|
| 問題明確 | 傳統 AOI 誤報率高達 30% |
| 技術驗證 | 已有 5 家客戶 POC |
| 量化效益 | 誤報率降至 3%，節省人力 70% |
| 市場規模 | 台灣 AOI 市場 100 億 |
| 國際競爭力 | 成本為進口設備 50% |

**計畫書摘錄**：
> 「台灣 PCB 產業為全球第一，但品質檢測仍高度依賴人工。傳統 AOI 設備誤報率高達 30%，導致產線人員需花費大量時間進行複檢，嚴重影響生產效率。
> 本計畫開發之 AI-AOI 系統，採用自研深度學習模型，經過 100 萬張瑕疵影像訓練，在 10 家 PCB 廠的驗證中，成功將誤報率從 30% 降至 3%，減少 90% 的複檢工作量...」

### 案例二：氮化鎵（GaN）電源模組

**公司背景**：
- 電源供應器 ODM 廠
- 成立 15 年，員工 80 人
- 既有產品線成熟

**計畫內容**：
> 開發高效能氮化鎵（GaN）快充電源模組，功率密度達業界領先水準

**成功要素**：
- 半導體新材料趨勢
- 技術團隊有相關經驗
- 明確的量化規格目標
- 已有客戶意向

**計畫書摘錄**：
> 「隨著 5G、電動車發展，高功率密度電源需求急增。GaN 元件較傳統矽元件可提升效率 5%、縮小體積 50%。
> 本計畫開發之 350W GaN 電源模組，功率密度達 30W/cm³（業界平均 15W/cm³），效率達 96%，可滿足伺服器、電動車充電等高功率應用需求...」

## 審查重點

### 電子電機類特殊審查標準

| 項目 | 審查重點 |
|------|----------|
| 技術規格 | 是否有明確可量測的規格目標？ |
| 國際水準 | 與國際領先技術的差距？ |
| 驗證數據 | 有無實測數據支持？ |
| 產業價值 | 對國內供應鏈的貢獻？ |
| 商業化 | 量產可行性？ |

### 評審可能問的問題

1. **技術面**
   - 「規格達成的可行性？」
   - 「與競品的技術比較？」
   - 「良率預估？」

2. **商業面**
   - 「客戶驗證狀況？」
   - 「量產成本結構？」
   - 「國際市場規劃？」

3. **產業鏈**
   - 「元件來源？」
   - 「代工/自製策略？」
   - 「認證規劃？」

## 計畫書撰寫技巧

### 強調規格量化

**範例**：

```
1.2 技術規格目標

┌─────────────────────────────────────────────────┐
│ 規格項目    │ 目前水準  │ 目標     │ 國際水準  │
├─────────────┼───────────┼──────────┼───────────┤
│ 效率        │ 92%      │ 96%      │ 95%       │
│ 功率密度    │ 15W/cm³  │ 30W/cm³  │ 25W/cm³   │
│ 工作溫度    │ 85°C     │ 105°C    │ 100°C     │
│ 待機功耗    │ 0.5W     │ 0.1W     │ 0.2W      │
└─────────────────────────────────────────────────┘
```

### 展示驗證數據

**範例**：

```
3.2 前期驗證成果

本計畫已完成以下驗證工作：

1. 電路設計驗證
   - 完成 Rev.A 電路板設計與打樣
   - 效率實測：94.2%（目標 96%）
   - 功率密度：22W/cm³

2. 可靠度測試
   - 高溫測試：通過 85°C/1000 小時
   - 電壓波動測試：±20% 輸入穩定輸出

3. EMI 初測
   - 傳導雜訊：MARGIN 3dB
   - 輻射雜訊：待優化

驗證結論：核心架構可行，Phase 2 將優化至目標規格
```

### 說明產業鏈貢獻

**範例**：

```
5.3 國內產業鏈貢獻

本計畫對國內半導體產業鏈的貢獻：

1. 技術自主化
   - 降低對國外 IC 依賴
   - 建立國內設計能力

2. 供應鏈在地化
   - 採用國內封測廠（預計 80% 在地化）
   - 帶動國內 PCB 廠訂單

3. 國際競爭力
   - 產品規格達國際水準
   - 成本具競爭力（較進口品低 30%）
```

## 經費編列注意事項

### 電子類特殊經費

| 項目 | 說明 | 備註 |
|------|------|------|
| 打樣費 | PCB、IC 打樣 | 需列明規格 |

| 設備租用 | 測試儀器 | 有合約 |

**Phase 2（1,000 萬元）**：
- 人事費：400 萬（40%）
- 材料/打樣：200 萬（20%）
- 設備：150 萬（15%）
- 委託測試：100 萬（10%）

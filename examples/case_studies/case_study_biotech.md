# 生技醫療產業 SBIR 案例指南

> 生技醫療是 SBIR 熱門領域，但也是審查最嚴格的產業之一。

## 產業概述

### 市場規模

| 指標 | 數據 |
|------|------|
| 台灣生技醫療產業產值 | 約 6,500 億元（2024） |
| 年複合成長率 | 8-10% |
| 上市櫃公司數 | 130+ 家 |

### SBIR 生技領域通過率

生技醫療類申請案通過率約 **25-30%**，略低於整體平均，原因：
- 技術門檻高
- 法規要求嚴格
- 商業化週期長

---

## 生技醫療領域分類

### 主要領域

| 領域 | 說明 | 適合階段 |
|------|------|----------|
| 新藥開發 | 小分子/大分子藥物 | Phase 2 以上 |
| 醫療器材 | Class I/II/III | 全階段 |
| 體外診斷 | IVD 試劑/設備 | 全階段 |
| 數位醫療 | AI/遠距/穿戴 | 全階段 |
| 保健食品 | 機能性食品 | Phase 1 |
| 再生醫療 | 細胞/基因治療 | Phase 2 以上 |

### SBIR 適合的細分領域

**最適合 Phase 1**：
- 醫材雛型開發
- AI 輔助診斷
- 遠距醫療軟體
- 體外診斷試劑

**最適合 Phase 2**：
- 醫材臨床前驗證
- 數位醫療平台
- 新劑型開發

## 成功案例分析

### 案例一：AI 醫療影像輔助診斷

**公司背景**：
- 新創公司，成立 3 年
- 團隊 12 人，含 3 名醫師顧問
- 前期已完成雛型開發

**計畫內容**：
> 開發肺部 X 光 AI 判讀系統，協助醫師快速篩檢肺結節

**成功要素**：

| 要素 | 做法 |
|------|------|
| 問題明確 | 引用國內肺癌篩檢數據 |
| 技術驗證 | 已有 1,000 例測試準確率 |
| 法規規劃 | 明確 TFDA 認證路徑 |
| 商業模式 | 與醫院的合作意向書 |
| 團隊專業 | 放射科醫師擔任顧問 |

**計畫書重點摘錄**：
> 「根據國健署統計，肺癌連續 10 年蟬聯國人癌症死因首位，每年新增個案超過 15,000 人，其中 70% 發現時已是晚期。早期發現是提高存活率的關鍵，但現行篩檢仰賴醫師肉眼判讀，每位放射科醫師每日平均需閱讀 100 份以上影像，容易疲勞漏診。
> 本計畫開發之 AI 系統，經過 50,000 張標註影像訓練，在前期 1,000 例驗證中達到 95% 敏感度和 92% 特異度，可在 10 秒內完成單一影像分析，協助醫師提早發現可疑病灶...」

### 案例三：快速檢測試劑

**公司背景**：
- 傳統試劑代理商轉型
- 首次申請 SBIR
- 申請 Phase 1

**計畫內容**：

**成功要素**：

- 團隊有試劑開發經驗
- 已有原料供應鏈
- 成本結構具競爭力

## 審查重點

### 生技類特殊審查標準

| 項目 | 審查重點 | 權重 |
|------|----------|------|
| 科學基礎 | 作用機制是否清楚？有無文獻支持？ | 高 |
| 技術可行性 | 前期數據是否支持？ | 高 |
| 法規途徑 | 認證路徑是否清楚？ | 高 |
| 臨床需求 | 是否解決真正的臨床問題？ | 中 |
| 商業模式 | 如何獲利？誰付錢？ | 中 |

### 評審可能問的問題

1. **技術面**
   - 「作用機制是什麼？」
   - 「有什麼前期數據？」
   - 「安全性考量？」

2. **法規面**
   - 「要申請什麼認證？」
   - 「預計認證時間？」
   - 「臨床試驗規劃？」

3. **商業面**
   - 「健保有給付嗎？」
   - 「誰是付費者？」
   - 「競爭產品有哪些？」

## 計畫書撰寫技巧

### 強調科學基礎

**範例**：

```
2.1 作用機制

本計畫開發之標靶藥物載體，基於以下科學原理：

1. 腫瘤微環境特性
   - pH 值較正常組織低（6.5-6.9 vs 7.4）[參考文獻 1]
   - EPR 效應增強奈米粒子累積 [參考文獻 2]

2. 智慧釋放機制
   - 在酸性環境觸發藥物釋放
   - 體外實驗顯示 pH 6.5 時釋放率 > 80%（圖 2-1）

3. 文獻支持
   - 類似策略已有 5 篇 Nature 子刊物發表
   - 本計畫創新在於 [具體差異點]
```

### 明確法規路徑

**範例**：

```
5.1 法規認證規劃

產品分類：第二等級醫療器材（Class II）
認證途徑：TFDA 查驗登記

時程規劃：
├─ M1-M6：設計開發與驗證
├─ M7-M12：送件準備
├─ M13-M18：TFDA 審查
└─ M19-M24：取證上市

所需文件：
- 技術文件（Technical File）
- 品質系統（ISO 13485）
- 電氣安全（IEC 60601-1）
- 生物相容性（ISO 10993）
- 臨床評估報告（CER）
```

### 展示臨床合作

**範例**：

```
6.2 臨床合作

本計畫已與以下醫療機構建立合作關係：

1. 台大醫院
   - 合作項目：臨床驗證 50 例
   - 合作醫師：OO科 XXX 教授
   - 合作意向書：附件 3

2. 榮民總醫院
   - 合作項目：回溯性數據分析
   - 資料量：3,000 筆病歷資料
   - IRB 已通過：附件 4
```

## 常見錯誤

### 錯誤一：忽略法規要求

**❌ 問題**
> 「產品開發完成後將申請 FDA 認證上市」（沒有任何規劃細節）

**✅ 修正**
> 詳細說明：
> - 產品分類依據
> - 認證途徑選擇
> - 所需文件清單
> - 時程規劃
> - 預算編列

### 錯誤二：缺乏科學依據

**❌ 問題**
> 「本產品具有抗氧化功能，可以抗老化」（無機制說明）

**✅ 修正**
> 說明：
> - 活性成分是什麼
> - 作用機制為何
> - 參考文獻出處
> - 前期驗證數據

### 錯誤三：商業模式不清

**❌ 問題**
> 「產品上市後將銷售給醫院」

**✅ 修正**
> 說明：
> - 目標客戶群
> - 定價策略
> - 通路規劃
> - 是否有健保給付
> - 競爭產品比較

## 經費編列注意事項

### 生技類特殊經費

| 項目 | 說明 | 備註 |
|------|------|------|
| 臨床試驗費 | 受試者費用、IRB 費用 | 需有試驗規劃 |
| 認證費用 | TFDA、ISO 等 | 需列認證項目 |
| 委託檢驗 | 安全性、效力測試 | 需指明檢驗項目 |
| GMP/GLP 費用 | 生產/實驗規範 | 需說明必要性 |

### 經費配置建議

**Phase 1（150 萬元）**：
- 人事費：60 萬（40%）
- 材料費：45 萬（30%）
- 委託檢驗：30 萬（20%）
- 其他：15 萬（10%）

**Phase 2（1,000 萬元）**：
- 人事費：400 萬（40%）
- 臨床試驗：200 萬（20%）
- 設備/材料：200 萬（20%）
- 認證費用：100 萬（10%）
- 其他：100 萬（10%）

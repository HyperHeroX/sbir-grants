# 地方型 SBIR 總覽

> 中央型 SBIR 補助金額高，但競爭激烈。地方型 SBIR 門檻較低，是中小企業的絕佳練兵場。

## 中央型 vs 地方型 SBIR

### 快速比較

| 項目 | 中央型 SBIR | 地方型 SBIR |
|------|-------------|-------------|
| 主管機關 | 經濟部中小及新創企業署 | 各縣市政府經發局 |
| Phase 1 上限 | 150 萬 | 100 萬 |
| Phase 2 上限 | 1,200 萬 | 100-200 萬 |
| 競爭程度 | 高 | 中 |
| 審查週期 | 較長 | 較短 |
| 適合對象 | 技術成熟、規模較大 | 入門、驗證構想 |
| 同時申請 | 可與地方型並行 | 可與中央型並行 |

### 策略建議

1. **新手入門**：先申請地方型 SBIR 累積經驗
2. **驗證構想**：用地方型做前期驗證，再申請中央型深化
3. **最大化資源**：符合條件可同時申請中央+地方

## 各縣市地方型 SBIR 一覽

### 台北市（SITI 產業發展補助）

> 台北市以「臺北市產業發展獎勵補貼及補助辦法」（SITI）提供類似補助

**基本資訊**
| 項目 | 內容 |
|------|------|
| 計畫名稱 | 臺北市產業發展補助計畫 |
| 主管機關 | 臺北市產業發展局 |
| 官方網站 | https://www.siti.gov.taipei/ |
| 補助上限 | 最高 300-500 萬 |

**申請資格**
- 依法登記於台北市的公司行號
- 符合中小企業認定標準
- 設立未滿 8 年的企業另有創業補助

**重點領域**
- 創新研發
- 品牌行銷
- 創業補助
- 數位轉型

**特色**
- 補助金額較高
- 涵蓋範圍廣（不限研發）
- 可補助品牌行銷費用

### 新北市

**基本資訊**
| 項目 | 內容 |
|------|------|
| 計畫名稱 | 新北市產業創新研發推動計畫 |
| 主管機關 | 新北市政府經濟發展局 |
| 官方網站 | https://www.economic.ntpc.gov.tw/ |
| 個別申請 | 最高 100 萬 |
| 聯合申請 | 最高 200 萬 |

**申請資格**
- 公司/商業/工廠登記於新北市
- 符合中小企業認定標準
  - 製造業：資本額 8,000 萬以下或員工 < 200 人
  - 其他業：營業額 1 億以下或員工 < 100 人
- 計畫執行場所須在新北市境內
- 每家公司每年限 1 案

**不符資格情形**
- 陸資投資企業
- 有政府計畫重大違約紀錄
- 欠繳稅捐
- 分公司不得申請

**審查重點**
- 創新性與技術可行性
- 計畫執行能力
- 預期效益

## 申請策略

### 策略一：先地方後中央

**適合對象**：首次申請、構想驗證階段

**執行方式**：
1. 先申請地方型 SBIR 做前期研究
2. 累積成果後申請中央型 Phase 1
3. 再申請中央型 Phase 2 擴大規模

**優點**：
- 降低風險
- 累積申請經驗
- 建立成果基礎

---

### 策略二：中央地方同步

**適合對象**：有經驗、計畫規模較大

**執行方式**：
1. 同時申請中央型與登記所在地的地方型
2. 計畫內容可相關但需有區隔
3. 注意不可重複補助相同項目

**優點**：
- 最大化資源取得
- 分散風險
- 台中市還有 10% 加碼

**注意事項**：
- 確認兩計畫內容不重複
- 經費不可重複請領
- 分開管理與結案

---

### 策略三：選擇最有利的地方

**考量因素**：

| 因素 | 說明 |
|------|------|
| 公司登記地 | 只能申請登記所在地的地方型 |
| 產業對應 | 選擇重點產業對應的縣市 |
| 加分項目 | 高雄有淨零/數位加分 |
| 競爭程度 | 人口少的縣市競爭較低 |

**建議**：
- 若公司登記可變更，考慮遷至有利縣市
- 選擇與公司產業對應的重點領域
- 留意各縣市特殊加分項目

## 申請時程建議

### 一般時程

| 階段 | 時間 | 工作內容 |
|------|------|----------|
| 準備期 | 1-2 個月 | 構想發想、資料收集 |
| 撰寫期 | 2-4 週 | 計畫書撰寫 |
| 送件 | - | 依公告截止日 |
| 審查 | 1-2 個月 | 書審、簡報 |
| 簽約 | 2-4 週 | 通過後簽約 |
| 執行 | 6-12 個月 | 計畫執行 |

### 注意事項

1. **收件時間**：各縣市不同，需留意公告
2. **全年受理**：部分縣市採隨到隨審
3. **梯次制**：部分縣市分梯次收件
4. **提早準備**：建議提前 2-3 個月開始準備

## 常見問題 FAQ

### Q1：可以同時申請多個縣市嗎？
**A**：不行。只能申請公司/商業/工廠登記所在地的地方型 SBIR。

### Q2：可以同時申請中央型和地方型嗎？
**A**：可以，但計畫內容需有區隔，經費不可重複請領。

### Q3：分公司可以申請嗎？
**A**：不行。多數縣市規定只有總公司可以申請。

### Q4：申請失敗可以再申請嗎？
**A**：可以。修正後可於下一梯次再申請。

### Q5：補助款何時撥付？
**A**：通常分期撥付，期中撥付 50%，結案後撥付 50%。

### Q6：已獲中央型補助，還能申請地方型嗎？
**A**：可以，但計畫內容需不同。台中市還有 10% 加碼優惠。
